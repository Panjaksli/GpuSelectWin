# GpuSelectWin
## Simple Windows utility to quickly set GPU mode for applications
### Features:
1) Can set GPU mode for:
 - Exe files
 - Windows shortcuts to exe files
 - Most of Steam games shortcuts
2) Doesn't run anything in background.
3) Is super light and non intrusive
### Usage:
Right click an exe/shortcut and select the desired GPU mode.

### Install:
Run the included exe file, it will install itself and add all necessary context menu entries.
### Uninstall:
Run the included file again, and it will delete all the necessary context menu entries. **Previously set GPU modes do not get reset!**
